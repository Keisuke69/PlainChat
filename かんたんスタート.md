# かんたんスタートガイド（非エンジニア向け）

このアプリを、ご自身のパソコンで動かすための手順です。コマンド操作はほとんど不要で、基本は **ファイルをダブルクリックするだけ** です。

---

## 用意するもの

1. **このアプリのフォルダ（`PlainChat`）** … 担当エンジニアから受け取るか、ダウンロードしておきます。
2. **Node.js（ノードジェイエス）** … アプリを動かすための無料ソフト。後述の手順で入れます。
3. **API キー** … OpenAI や Anthropic（Claude）の API キー。お持ちのものを使います（取得方法は社内の担当者にご確認ください）。

---

## はじめての起動（最初の1回だけ少し時間がかかります）

### Mac の場合

1. `PlainChat` フォルダを開きます。
2. **`start.command`** をダブルクリックします。
   - 「開発元を確認できないため開けません」と出たら、`start.command` を **右クリック →「開く」** を選び、確認ダイアログで「開く」を押してください（最初の1回だけ）。
3. 黒い画面（ターミナル）が開きます。**Node.js が入っていない場合**は自動でダウンロードページが開くので、**「LTS」** と書かれた方をインストールしてから、もう一度 `start.command` をダブルクリックしてください。
4. しばらく待つと（初回は数分）、**自動的にブラウザが開いて** アプリが表示されます。

### Windows の場合

1. `PlainChat` フォルダを開きます。
2. **`start.bat`** をダブルクリックします。
   - 「WindowsによってPCが保護されました」と出たら、「詳細情報」→「実行」を押してください（最初の1回だけ）。
3. 黒い画面が開きます。**Node.js が入っていない場合**は自動でダウンロードページが開くので、**「LTS」** をインストールしてから、もう一度 `start.bat` をダブルクリックしてください。
4. しばらく待つと（初回は数分）、**自動的にブラウザが開いて** アプリが表示されます。

> ブラウザが自動で開かないときは、ブラウザのアドレス欄に **`http://localhost:3000`** と入力してください。

---

## アプリの使い方

1. 最初の画面で **「アカウントを作成する」** を押し、名前・メールアドレス・パスワード（8文字以上）を入れて登録します。
   - 2回目以降は同じメールアドレスとパスワードでログインします。
2. 画面左下の **「設定」** を開き、**OpenAI / Anthropic の API キー** を貼り付けて「保存」します。
   - キーは暗号化して保存され、画面には末尾4桁だけ表示されます（安全のため全体は表示されません）。
3. チャット画面に戻り、上部のプルダウンで **使いたいモデル** を選びます。
4. 右側のパネルで **システムプロンプト** や **パラメータ** を調整できます（モデルによって表示される項目は変わります）。
5. 下の入力欄にメッセージを入れて **送信**。やりとりは自動で保存され、左側の一覧からいつでも見返せます。

---

## 終了するには

- 黒い画面（ターミナル）で **Mac は `Control + C`**、**Windows は `Ctrl + C`** を押すと止まります。そのままウィンドウを閉じてかまいません。

## 2回目以降の起動

- 同じように **`start.command`（Mac）/ `start.bat`（Windows）をダブルクリック** するだけです。2回目以降はすぐに起動します。

---

## 困ったときは

- **ブラウザに何も表示されない** → 黒い画面に「起動しました」と出るまで少し待ってから、ページを再読み込みしてください。
- **「API キーが未設定です」と出る** → 「設定」画面でキーを保存できているか確認してください。
- **うまく動かない** → 黒い画面に表示されているメッセージをそのまま担当エンジニアに伝えてください。

> 補足: このアプリはあなたのパソコンの中だけで動きます（データは外部に送られません。LLM への問い合わせ内容のみ、選んだプロバイダーの API に送信されます）。
